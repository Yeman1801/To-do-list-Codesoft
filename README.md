# Project Title

To-Do List with Time Tracker (Tkinter)

---

## Description

This is a Python-based GUI application developed using Tkinter.  
The application allows users to create a to-do list by adding tasks along with start and end times.  
It automatically calculates the duration of each task and displays it in a structured table format.

## Features

- Add task with start time and end time
- Automatic time duration calculation
- Auto-format time input (HH:MM)
- Input validation with popup messages
- Delete selected tasks
- Clean and simple graphical interface

---

## Technologies Used

- Python 3
- Tkinter
- ttk
- datetime module

## Usage

Run the application:
- python main.py

Enter the task name.

Enter start time and end time (HHMM format).

Click Add Task.

View the task duration in the table.

Select a task and click Delete Task.

## Validation

- All fields are required

- End time must be greater than start time

- Invalid inputs will show error messages
